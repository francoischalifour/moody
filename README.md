# Moody

> Moody is a web app that plays music according to your mood.

*Built with: ES6, React, SpotifyAPI.*

## Dev

To build the project, you will first need to install [npm](http://npmjs.org).

###### Commands

- Init: `$ npm install`
- Run: `$ gulp watch`
- Browse: `http://localhost:9000/`
